The following applies to the result file "getdata-013-CP-tidy_data.txt"

"Subject": int
--------------
Each value represents one of the 30 subjects of the experiment.
values: 1,2,3,..,30


"Activity": Factor(6)
---------------------
Each value represents the type of activity performed
values: 
"WALKING"
"WALKING_UPSTAIRS"
"WALKING_DOWNSTAIRS"
"SITTING"
"STANDING"
"LAYING"
